# Greedy-Snake (贪吃蛇)
Use Typescript to edit this project(使用 Typescript 编写这个项目).



## Project setup (项目依赖安装)

```bash
npm install
```



### Compiles and hot-reloads for development (开发中启动)

```bash
npm run serve
```



### Compiles and minifies for production (项目打包,可部署)

```bash
npm run build
```
